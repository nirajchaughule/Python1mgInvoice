This file is created using AI tools which include CHATGPT and Deepseek R1,
this was created to solve a personal problem of manually going through the Inbox to find value of SubTotal for each invoice as a invoice PDF was not provided by the sender.
The code reads through the email, converts the email to PDF where a SubTotal is present, extracts the Data to an excel with the filename and amount for each PDF.
The Apppassword for gmail has to be setup first if you use a 2FA for gmail, else you can use your gmail password.
Even if you use the Apppassword or your gmail password, ensure that you remove it from the script after a successful run to avoid any security issues.
